# defAI — Boss Fight Demo (WebGL)

A shareable WebGL build of the current boss fight from **defAI**, for playtesting.

**▶ Play:** https://michyeohey.github.io/defai-boss-demo/

## Controls
- **Move:** WASD (or left stick)
- **Aim:** move the mouse (or right stick) — the character faces your cursor
- **Shoot:** left mouse button (or right trigger)
- **Start the fight:** press **SPACE**

Boss health (top) and your health (bottom-left) are shown on the HUD. Kill the boss
before the song ends — if it finishes first, the boss enrages and wipes you.

*Loads ~95 MB the first time (engine + assets), so give it a moment. Chrome/Edge/Firefox.*
